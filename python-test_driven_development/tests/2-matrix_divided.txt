How to Use 2-matrix_divided
=============================

Import the function:
    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Test normal matrix with integers:
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Test if original matrix is modified (it shouldn't be):
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> _ = matrix_divided(matrix, 3)
    >>> matrix
    [[1, 2, 3], [4, 5, 6]]
